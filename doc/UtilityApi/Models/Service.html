Overview

- (Array<Range<Time>>) bill_coverage

A list of date ranges that are covered by collected bills. If we have a complete history of bills, there will be only one date range in this array. If there is are some missing bills, the date ranges will only show the coverage that is covered by the collected bills. If no bills have been collected, this array will be empty. This field can be useful in checking to see if the Service meets your coverage requirements.
